The literal 10000000000 of type int is out of range

2019-11-05


出现上述报错的主要原因是：

Java中默认整数为int类型，虽然图片中的数字未超过long类型的最大值，但由于默认整数为int类型，该数超过了int类型的最大值，所以报错；

解决方法如下：

在声明时，在数字的后面添加l或者L即可，建议添加L，更容易区别
